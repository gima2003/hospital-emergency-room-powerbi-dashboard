🏥 Hospital Emergency Room Dashboard
Project Overview

This project presents an interactive Power BI dashboard designed to analyze Emergency Room (ER) operations and patient management data.

The dashboard enables healthcare stakeholders to monitor patient flow, waiting times, referrals, satisfaction levels, admissions, and demographic trends to support data-driven decision making.

Business Requirements

The dashboard was developed to track:

Number of Patients
Average Wait Time
Patient Satisfaction Score
Number of Patients Referred

Additional analysis includes:

Admission Status
Age Group Distribution
Gender Distribution
Race Distribution
Department Referrals
Patients Seen Within Target Time
Patient Traffic by Day and Hour
Dashboard Views
1️⃣ Monthly View

Provides month-by-month performance analysis including:

Total Patients
Average Wait Time
Patient Satisfaction
Referral Trends
2️⃣ Consolidated View

Provides overall hospital performance across the entire reporting period.

3️⃣ Patient Details View

Allows detailed exploration of individual patient records.
