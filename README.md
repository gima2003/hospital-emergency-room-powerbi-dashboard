# 🏥 Hospital Emergency Room Dashboard

## 📖 Project Overview

This project presents an interactive **Power BI Dashboard** designed to analyze Emergency Room (ER) operations and patient management data.

The dashboard helps healthcare stakeholders monitor:

- Patient flow
- Waiting times
- Patient satisfaction
- Admissions
- Department referrals
- Demographic trends

The goal is to support **data-driven decision making** and improve operational efficiency within the emergency department.

---

## 🎯 Business Requirements

The dashboard was developed to track the following KPIs:

- 👥 Number of Patients
- ⏱️ Average Wait Time
- ⭐ Patient Satisfaction Score
- 🏥 Number of Patients Referred

### Additional Analysis

- Admission Status
- Age Group Distribution
- Gender Distribution
- Race Distribution
- Department Referrals
- Patients Seen Within Target Time
- Patient Traffic by Day and Hour

---

## 📊 Dashboard Views

### 1️⃣ Monthly View

Provides month-by-month performance analysis including:

- Total Patients
- Average Wait Time
- Patient Satisfaction Score
- Referral Trends

### 2️⃣ Consolidated View

Provides an overall summary of hospital performance across the entire reporting period.

### 3️⃣ Patient Details View

Allows detailed exploration of individual patient records and operational information.
